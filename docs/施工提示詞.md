# 施工提示詞集：要做哪項，複製對應提示詞貼上即可

> **使用方法：**
> 1. 開新對話（建議先用 `/model` 切到 Sonnet，省額度）
> 2. 找到你要做的任務，把整段提示詞複製貼上
> 3. 照 AI 的步驟走。它會自動讀到 CLAUDE.md 的規則（不會亂部署、完工會給你驗證清單）
>
> **建議順序：1 → 2 → 3 → 4 → 5 → 6，一次做一項，做完驗證通過再做下一項。**

## 任務 5：點數到期提醒推播（自動）

```
我要做「方案快到期自動提醒」功能：每天自動檢查所有客戶，找出「7 天內到期、而且還有剩餘次數」的方案，透過 LINE 推播提醒客戶回來預約。

背景：我的系統是 index.html（後台）＋ booking.html（客戶端）＋ booking-worker.js（Cloudflare Worker，已有 LINE 推播能力，pushSecret 存在 Firestore config/app）。方案資料含 expiryDate、totalSessions、usedSessions 欄位。

請先提出計畫給我看（這是新功能，照 CLAUDE.md 的 C 級流程），計畫要涵蓋：
1. 建議用 Cloudflare Worker 的 Cron Trigger（wrangler.toml 加排程）每天固定時間跑，如果你有更好的做法請說明比較
2. 提醒訊息文案（親切、繁體中文，署名店家，包含剩餘次數和到期日）
3. 防重複機制：同一個方案不要每天都提醒，最多提醒 1-2 次
4. 只推播給有綁定 lineUserId 的客戶，沒綁定的整理成清單讓我人工聯絡

我同意計畫後才動工。完工後 wrangler deploy 指令準備給我自己執行，並給我驗證方法（例如先用我自己的測試帳號做一個明天到期的假方案試推）。
```
